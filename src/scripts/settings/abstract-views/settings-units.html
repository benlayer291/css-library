Default settings for the following properties can be set using previously defined variables:

  • base-font-family
  • base-font-size
  • base-font-weight
  • base-line-height
  • base-spacing-unit
  • base-text-color
  • base-background-color
/*
  Define high-level variables for the site
*/

$base-font-family:        $family-sans-serif !default;
$base-font-size:          $font-size-6 !default;
$base-font-weight:        $weight-normal ! default;
$base-line-height:        ($base-font-size * 1.5) !default;
$base-spacing-unit:       ($base-font-size * 1.5) !default;
$base-text-color:         $grey-darker !default;
$base-background-color:   $white !default;
