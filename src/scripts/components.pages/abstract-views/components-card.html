A card is simple component useful for displaying items, especially when used in conjunction with grids

The header, body and footer sections of the card are containers for any type of content

The image section can have various ratios applied to it by adding to or modifying the existing image--is-4by3 modifier

An example card footer 
<div class="card">
  <div class="card__image">
    <figure class="image image--is-4by3">
      <img src="http://placehold.it/300x225" alt="4x3 placehold image">
    </figure>
  </div>
  <div class="card__header">
    An example card header
  </div>
  <div class="card__body">
    An example card content
  </div>
  <div class="card__footer">
    An example card footer
